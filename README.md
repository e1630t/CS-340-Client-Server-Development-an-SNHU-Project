# CS-340-Client-Server-Development-an-SNHU-Project
This project was developed as part of a CS 340 course assignment designed to give experience on creating reusable Python code that interacts with a NoSQL database by implementing Create, Read, Update, Delete operations.
This CRUD module then served as the foundation for developing a dashboard for user interfacing. Which gave additional experience in dashboard development and data visualization.
